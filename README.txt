Step 1: Unzip the downloaded file ‘app’ into your WAMP/MAMP webroot.
Step 2: Start WAMP/MAMP
Step 3: Open Visual Studio Code
Step 4: Go to Phpmyadmin check in the top bar, MySQL Port number (e.g. 3306, 3308, etc.) and note this down
Step 5: Go open Visual Studio Code and navigate to app/include/ConnectionManager.php and verify your port number if your port number is different replace it to your own port number
Step 6: Go open app/database/create.sql
	Load this set of SQL statements in phpmyadmin
